Tablacus Context Menu
    • コマンドラインからコンテキストメニューを表示・実行します。

    Usage:
    tcm64 [options] 項目 [項目2]…
    • 64ビットのコンテキストメニューを実行
    tcm32 [options] 項目 [項目2]…
    • 32ビットのコンテキストメニューを実行

    Options:
    -xコンテキストメニューを表示する x座標 省略時はマウスカーソルのx座標
    -yコンテキストメニューを表示する y座標 省略時はマウスカーソルのy座標
    -Directory作業ディレクトリ 省略時は NULL
    -Parameterパラメータ 省略時は NULL
    -ShowSW_ values 省略時は SW_SHOWNORMAL
    -fMaskCMIC_MASK_ values
    -FlagsOptional flags 省略時は 0 (CMF_NORMAL)
    -Timerコマンド実行後にTablacus ContextMenuが終了するタイマー 省略時は 1
    -Invoke名前のコマンドを実行
    -Drop右ボタンのドラッグ＆ドロップメニューを表示
    -grfKeyStateドラッグ＆ドロップのボタンステート

    e.g.:
    tcm32 -x 500 -y 200 C:\hoge\fuga c:\hoge\piyo
    tcm64 -x 500 -y 200 C:\hoge\fuga c:\hoge\piyo
    tcm64 -Invoke "*Microsoft Security Essentials*" C:\hoge\fuga
    tcm32 -Invoke "*Microsoft Security Essentials*" C:\hoge\fuga
    tcm64 -Drop C:\hoge\fuga C:\fuga.txt
    tcm32 -Drop C:\hoge\fuga C:\fuga.txt
注意事項
  • 項目は同一フォルダで指定してください
  • 有効な項目を指定しなかった場合はTablacus ContextMenuのウインドウが表示されます。項目をドラッグ＆ドロップするとドロップした項目のコンテキストメニューを表示します。
