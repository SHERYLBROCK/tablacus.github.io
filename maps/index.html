Basic replace

From: Google Maps API URL
https://maps.googleapis.com/maps/api/js?key=YOUR_API_KEY&callback=initMap

To: Tablacus Maps API URL
https://unpkg.com/tablacusmapsapi/tablacusmapsapi.js?alias=google&callback=initMap
