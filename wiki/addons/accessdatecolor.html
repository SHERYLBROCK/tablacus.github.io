Colored with access date

Function

Display the item name in specific color if an item is accessed in a specific elapsed time

Note

  • If you check Explorer browser, this add-on will not work.
  • Elapsed time can be specified in seconds, minutes, hours, days, weeks or years, but the actual calculation will be in seconds.
  • Example: “1d” will match any item modified in the past 86400 seconds.
  • 1 year is assumed 365 days. No leap year handling.

Config file

accessdatecolor.tsv
