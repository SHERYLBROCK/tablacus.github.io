Add-on switcher

Function

Toggle the on/off state of another add-on by a hot key, a mouse gesture, a command button or a menu option

Note

  • Useful when you need to toggle an add-on frequently but this ability is not built-in.
  • Example – to add a tool bar button to toggle Side Tree View:
    1. Choose “Tools > Add-ons”
    2. Click “Options” of “Tool bar”
    3. Add a button with:
  • Name: Folder Tree
  • Type: Addon switcher
  • Options: sidetreeview (click Browse... to choose)
  • Icon: choose one you like (click Browse... to choose)
