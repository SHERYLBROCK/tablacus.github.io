Clip folder

Function

A clip folder is a virtual folder that can hold any files and folders from anywhere in the computer. It is similar to a regular folder that stores shortcuts, except that the target files and folders are presented as themselves.

Note

  • A clip folder is actually a text file encoded in UTF-8 with BOM, and has an extension of cfu.
  • Right click an empty space of a folder to create a clip folder. Alternatively you can use any application to create a cfu file.
  • Click a cfu file to open it like a regular folder.
