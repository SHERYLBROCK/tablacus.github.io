Count bar

Function

Display both item count and selection count

  • Typically used as part of a status bar info line.
  • Instantly shows number of all kind of elements (files, directories, volumns, ...) displayed in active detail view.
  • Additionaly shows instantly the number of selected elements there.

Note

You can opt to display on the title bar. Then you also need to:

  • Set the location to “outside of the TE window”
  • Disable the “title bar” add-on
