Disable Tree Drop

Function

Disable drag-and-drop to the folder tree

  1. from another tree node, and
  2. from a file list, and
  3. from another application

Note

  • “Disable Drag-and-Drop” option in “Tools > Options > Tree” can help case 1 only. Use this add-on to achieve all 3 cases.
  • Drag from the folder tree to outside (to the tab area to open a folder, for example) is not affected.

Japanese

ツリーのドロップ禁止

ツリーに対するドロップ操作を出来なくする。
