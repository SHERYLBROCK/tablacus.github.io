Download

Function

Download files with external apps. (When XMLHttpRequest does not work etc.)

Environment variables Description
%url% Download URL
%file% Destination file

Note

  • Usually this add-on is not needed. TE uses a Windows built-in function to do the download.
  • Click a button on the Options page to choose an application. Then amend the path of cURL / Wget. No need to amend the path of PowerShell and BITSAdmin which are Windows built-in.
  • To show trouble-shooting messages during download, opt for “Visible”.
