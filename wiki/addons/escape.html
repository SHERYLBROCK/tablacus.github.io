Escape Unicode

Function

Command button to check and rename any item in the current folder for Unicode characters

  • Escape: Rename away from Unicode
  • Unescape: Reverse the renaming

Note

  • TE will ask you to confirm the new names before the operation.
  • Useful if you have trouble accessing these items, especially when you transfer them to a different file system that does not support Unicode.
