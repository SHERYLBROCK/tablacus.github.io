Horizontal Favorites bar

Function

Tool bar of the favorites items

  • Click an item: Open the item
  • Right click an item: Context menu
  • Drop to an item: File copy/move the dragged source to the item
  • Drop to an empty space of the tool bar: Add the dragged source to the Favorites menu

Note

  • At menu 'Favorites' you can add any directory location to your favorites list.
  • To quickly jump to this locations the Horizontal Favorites Bar presents them as buttons .
  • A right click onto a favorite gives the original context menu
