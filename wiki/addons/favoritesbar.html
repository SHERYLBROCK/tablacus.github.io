Favorites bar

Function

Vertical tool bar of the favorites items shown one on each line

  • Click an item: Open the item
  • Right click an item: Context menu
  • Drop to an item: File copy/move the dragged source to the item
  • Drop to an empty space of the tool bar: Add the dragged source to the Favorites menu
