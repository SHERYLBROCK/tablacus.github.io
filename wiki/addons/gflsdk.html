GFL SDK

Function

Extend the image using GFL SDK.

Note

  • TE32: 32bit dll are required.
  • TE64: 64bit dll are required.

Another add-on or [application] required

  • [GFL SDK v3.40 - XnView Software]

Optional

  • Thumbnail plus : Extended thumbnail image view.

Option

Set up

Click "Get GFL SDK..."

Click "http://download.xnview.com/GflSDK-win.zip"

Extract the ZIP file to any folder

Click "Browse.." and select dll.

  • 32bit libfl*.dll
    • GfkSDK\libW\libgfl340.dll
  • 32bit libfle*.dll
    • GfkSDK\libW\libgfle340.dll
  • 64bit libfl*.dll
    • GfkSDK\lib64W\libgfl340.dll
  • 64bit libfle*.dll
    • GfkSDK\lib64W\libgfle340.dll
