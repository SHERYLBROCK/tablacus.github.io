Icon Changer

Function

  1. Display another icon for:
    • opened folder
    • closed folder
    • all undefined file types
    • a specific file type
  2. Display another icon overlay for:
    • shortcut
    • network share

Note

  • “Undefined” means that the file type is not associated with any application in Windows.
  • If a specific file type is added to the configuration list of this add-on, it must have been associated with an application. Otherwise nothing will happen.
