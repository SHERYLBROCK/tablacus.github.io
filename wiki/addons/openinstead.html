Open Instead

Function

Automatically open the folder in a new tab when it is opened in Windows Explorer. The Explorer window is then closed.

Note

  • To use, in the configuration page of the add-on:
    • Tick “Real Folders”
    • Tick “Special Folders”
    • Tick “Take over the Explorer view” (if necessary)
  • “Take over the Explorer view” option lets TE take in a folder together with its current view in Windows Explorer, but the following Folder View add-ons must not be enabled:
    • Take over folder view settings
    • Folder settings
