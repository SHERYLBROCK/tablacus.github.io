Prevent Duplicate Tabs

Function

Automatically prevent from any duplicated tab in the same pane

Note

  • When a tab is added (by clicking a “new tab” button, for example), the “Recent Items” folder will be opened, instead of the default of a duplicate of the current folder.
  • When a folder is chosen to open (by choosing it from the Favorites menu, for example) and is found duplicated, it will not be opened.
  • When you navigate to a folder which is already opened in another tab, the tab will be closed.
  • Prevention works on pane basis. The same folder can still be opened or be navigated to on another pane.
