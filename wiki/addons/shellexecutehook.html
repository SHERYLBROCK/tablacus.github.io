Shell execute hook

Function

  1. Run TE instead of Windows Explorer when:

    • you press Win+E
    • you run Windows Explorer from:
      • Start Menu
      • Task Bar (when pinned)
      • Run dialog box (Win+R)
      • Command Prompt
    • an application runs Windows Explorer

  2. Run TE to open the folder in a new tab instead of Windows Explorer when:

    • you input a folder path in the Run dialog box
    • you input “start folder path” in Command Prompt
    • an application opens a folder in Windows Explorer

Note

  • This add-on uses the registry.
  • TE needs not be running when it is invoked this way.
  • You can still run Windows Explorer by other ways such as:
    • choose “Computer” from Start Menu
    • click Recycle Bin on Desktop
    • click a folder shortcut on Desktop or Task Bar
    • choose “Open in Explorer” from tab context menu
  • Running TE from another application may or may not work, depending on the technique used by that application.
  • You can use “Open Instead” add-on to handle cases this add-on misses.
  • Steps to use (administrator privilege is needed):
    1. In the configuration page of the add-on:
      • tick all “Registry” options
      • click the “Default” button and reply OK
      • tick “Windows Explorer”
      • click OK
    2. Your desktop will refresh to complete the configuration.
      • If you do not tick “Windows Explorer”, you will need to restart your computer to complete the configuration.
