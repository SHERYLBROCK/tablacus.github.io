Speed Dial

Function

Show a list of most frequently opened folders when you add a tab

Note

  • This speed dial list becomes the default new tab.
  • A maximum of 9 folders are listed.

Config file

speeddial.tsv

Japanese

スピードダイヤル

新しいタブに、頻繁にアクセスしたタブのリストを表示して素早く開けるようにする。

(オプション＞タブ＞新しいタブの設定よりも、このアドオンが優先される)
