Stripes lite

Function

Display alternating stripes in the background in details view

Note

Stripes may be missing from a sort column. Use “Color” or “No emphasis” add-on to solve the problem.

Please install an add-on No emphasis if the sort column is inappropriate like this.

If you check Explorer browser, this add-on will not work.
