Change Tab Color

Function

Context command to set a color to a tab, i.e. a specific folder

Config file

tabcolor.xml

Another add-on or [application] required

  • Tab plus

Simular Add-ons

  • Tab color plus

Japanese

タブに色を付ける

  • タブに任意の色を付ける。(タブに個別の色を付ける時に使う)
  • タブを閉じても色はキャンセルされないので、同じフォルダを開くと色は残っている。
