Tab design

Function

Customize the appearance of tabs

  • 4 predefined styles are available
  • Advanced users can specify their own (in CSS syntax)

Usage

Tabs Types

  • Default - Base styles for all tabs
  • Selected Items - Currently active tab
  • Left - Tabs to the left of the currently active tab
  • Right - Tabs to the right of the currently active tab
  • New tab - Newly created tabs
  • Base - Background for the tab bar area

Style

  • Default - Style for the current tab type
  • Add Before - Content to be at the beginning of the tab content
  • Add After - Content to be at the end of the tab content

Another add-on or [application] required

  • Tab plus
