Tab groups

You can manage many open tabs.

Function

A workspace (tab group) is a collection of of tabs and panes you work with. This add-on lets you open several workspaces at the same time, and switch among them by clicking the workspace tabs.

Note

  • Each workspace can have its set of tabs and pane layout.
  • You can save a workspace as a file and load it later.
  • You can drag-and-drop objects among the workspaces.
  • TE saves all opened workspaces when it exits, and restores them when it starts.
