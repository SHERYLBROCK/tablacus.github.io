Tabs

Function

Enriched tab functions:

  1. Show folder path in tip popup
  2. Enable tab arrangement within a pane and among panes
  3. Open a folder in a new tab by drag-and-drop to the tab area
  4. Enable file copy and move operations by drag-and-drop items to another tab

Note

To customize the appearance of tabs, choose "Tools > Options > Tabs".
