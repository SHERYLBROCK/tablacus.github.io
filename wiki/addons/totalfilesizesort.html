Total file size sort

Function

Command button to sort all items by “Total File Size” in descending order, and display the column if not yet.

Note

  1. “Total File Size” is:
    • for a folder: total size of all files under the folder branch
    • for a file: same as its size
  2. This sorting order cannot be preserved in folder view.
  3. Require Window Vista or higher
