Total Commander Lister Plug-in

Function

View a selected file in a new tab with

  • Hot key Ctrl+Q
  • Context command “File Content”
Supported file type Plug-in
Image Imagine
Video, audio mmedia
HTML, XML HTMLView
others others

Note

Plug-in installation

  1. Download a plug-in from the internet
  2. Unpack the ZIP/RAR file
  3. Add the plug-in to TE using the Options page.
    • Some informational check boxes will be ticked.
    • In TE64, choose a WLX or UWLX file and let TE detect the WLX64 file.
  4. Repeat step 1 to 3 for each required plug-in
    • Total Commander the application is not required.

Config File

wlx.xml

Another add-on or [application] required

[Total Commander Lister Plug-in]
